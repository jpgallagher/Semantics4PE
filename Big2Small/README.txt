Code and examples for big-to-small step semantics transformation.

smallStepSolve.pl - the interpreter for big-step rules
smallStepSolve.pl.ann - the Logen-annotated interpreter
normalise.pl - normalise a set of big-step rules
clp2logen.pl - translate rules to form needed for Logen
big2small.sh - shell script for performing transformation

Examples 

bigstep_while.pl - simple imperative language
bigstep_lambda.pl - call-by-value lambda-calculus
bigstep_mini_ml.pl - Mini-ML semantics by G. Kahn
